Known Issues

Not all known bugs are published: often bugs are resolved quickly or do not affect users.

  • Search functionality in the FISMA system inventory report doesn't report back when searching the word "Google". This is a known issue due to the fact the Related Artifact field has google hyperlinks in them.

  • Application Interface Diagram - spacing between each application is limited causing the user to pull apart applications manually.

  • Internet Explorer (IE) compatibility with GEAR - unable to export reports.

  • The browser "Back" button will take you back to the previously viewed page, but will not retain any report selections or filters applied.
